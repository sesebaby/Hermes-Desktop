## 任务陈述

基于 `external/hermescraft-main` 的“一 NPC 一常驻 agent”思路，为当前 Hermes-Desktop 制定增强方案：每个 Stardew NPC 都应成为独立常驻运行体，拥有自己的长期记忆、SOUL、会话和自主循环；同时比参考项目更完善，补齐收件箱、租约、资源短占位、任务恢复和事件序号流。

## 期望结果

- 每个启用的 NPC 都有自己的常驻 Hermes agent 循环，而不是由一个中心 `foreach` 轮流 tick。
- 每个 NPC 继续使用独立 namespace：`SOUL.md`、memory、transcripts、activity、traces。
- 私聊控制权与自治行动控制权显式分离，不再靠单一“当前任务”粗暴串行。
- 共享世界冲突采用轻量机制处理：会话租约、行动槽位、资源短占位、可恢复后台任务。
- 保持当前仓库约束：仍由 Desktop 单进程托管多个 NPC runtime，不改成“一 NPC 一 OS 进程”。

## 已知事实 / 证据

1. 当前 Desktop 已经具备 NPC 独立 namespace 基础。
   - `src/runtime/NpcNamespace.cs`
   - `src/runtime/NpcRuntimeDescriptorFactory.cs`
   - `src/runtime/NpcRuntimeContextFactory.cs`

2. 当前 NPC runtime 已经具备独立 context 装配链：
   - `SoulService`
   - `MemoryManager`
   - `TranscriptStore`
   - `TranscriptRecallService`
   - `ToolRegistry`
   - 证据：`src/runtime/NpcRuntimeContextFactory.cs`

3. 当前自治循环仍然是中心后台服务轮询所有 NPC，而不是每个 NPC 自己持有常驻循环。
   - `src/games/stardew/StardewNpcAutonomyBackgroundService.cs`
   - `Desktop/HermesDesktop/App.xaml.cs`

4. 当前私聊已经有比较成熟的租约暂停机制。
   - `src/runtime/NpcRuntimeInstance.cs`
   - `src/game/core/PrivateChatOrchestrator.cs`
   - `src/games/stardew/StardewNpcPrivateChatSessionLeaseCoordinator.cs`

5. 当前 `NpcRuntimeInstance` 还没有“每 NPC 自己的收件箱 + 驱动任务 + 行动状态机”。
   - `src/runtime/NpcRuntimeInstance.cs`
   - `src/runtime/NpcRuntimeSupervisor.cs`

6. 参考项目的竞争处理主要依赖：
   - 每 agent 独立 home / 进程 / body server
   - bot 内部只允许一个后台任务
   - 点名路由 + command queue
   - 社会冲突自处理
   - 证据：
     - `external/hermescraft-main/civilization.sh`
     - `external/hermescraft-main/bot/server.js`
     - `external/hermescraft-main/bot/lib/chat.js`
     - `external/hermescraft-main/SOUL-civilization.md`

7. 参考项目自己承认当前方案仍偏原型：
   - 单后台任务
   - 以前存在后台任务覆盖问题
   - 依赖轮询，没有实时事件推送
   - 证据：
     - `external/hermescraft-main/docs/archive/AUDIT-REPORT.md`
     - `external/hermescraft-main/docs/archive/HACKATHON-PLAN.md`

## 约束

- 计划名称和内容必须用中文。
- 不回退用户已有脏改动。
- 不新增第二套 Desktop/NPC prompt 装配主链。
- 不把方案改成“一 NPC 一 OS 进程”。
- 必须严格保留“桌面 agent 和 NPC 本质等同”的方向，只允许 namespace / persona / session / body 侧差异。
