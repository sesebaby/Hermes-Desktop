## Reference Chain Reconstruction

| Layer | Reference behavior | Reference anchors | Current mapping target |
| --- | --- | --- | --- |
| Trigger | The model calls the single `todo` tool. Omitting `todos` reads; providing `todos` writes. | `external/hermes-agent-main/tools/todo_tool.py`; `run_agent.py` todo dispatch around `function_name == "todo"` | `src/Tools/TodoWriteTool.cs`; `src/Core/Agent.cs` |
| Snapshot | One in-memory todo store per agent/session. Items are ordered and statuses are `pending`, `in_progress`, `completed`, `cancelled`. | `TodoStore` in `tools/todo_tool.py` | `src/tasks/SessionTodoStore.cs` keyed by session id |
| Prompt / Summary Assembly | Active pending/in-progress todos are injected after compression; completed/cancelled todos are filtered out. | `TodoStore.format_for_injection`; `run_agent.py` compression injection | `ContextManager.PrepareContextAsync` -> `PromptBuilder.ActiveTaskContext` |
| Parse / Repair / Normalize | Missing id becomes `?`; missing content becomes `(no description)`; invalid status becomes `pending`; duplicate ids keep last occurrence at its final position. | `_validate`; `_dedupe_by_id`; reference tests | `SessionTodoStore.Validate`; `DeduplicateById` |
| Projector / Executor | Tool execution mutates the store and always returns the full current list plus summary. | `todo_tool`; `run_agent.py` sequential and concurrent tool paths | `TodoTool.ExecuteAsync`; `Agent.ExecuteToolCallAsync` |
| Authoritative Writeback | The full tool result is the source of truth. TUI ignores `tool.start` args for todos because merge args may be partial. | `tui_gateway/server.py` `_on_tool_start` and `_on_tool_complete` | `TranscriptStore` observer -> `SessionTaskProjectionService.OnMessageSavedAsync` |
| Player-visible Surface | TUI records full todos on `tool.complete`, shows a live todo panel, and archives the turn's todos into a transcript trail on message completion. | `turnController.recordTodos`; `turnStore.archiveTodosAtTurnEnd`; `todoPanel.tsx` | `SessionTaskPanelModel`; `TaskPanel.xaml.cs`; possible future transcript archive |
