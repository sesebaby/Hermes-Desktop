# Stardew Navigation Index

Use this file only to choose which regional reference to load next. It intentionally does not contain mechanical coordinates.

## Regions

- Beach: shoreline, pier, tide-pool side, and south exits. Load `references/regions/beach.md`.
- Town: square, fountain, shops, clinic path, and public gathering points. Load `references/regions/town.md`.

## Rule

Do not output `target(locationName,x,y,source)` from this index. Load the relevant region file, then a POI file.
