# 鬼谷八荒参考资料索引

本目录用于沉淀鬼谷八荒与本项目最相关的官方资料、社区教程和开源线索。

当前重点围绕三类能力：

- 对话与剧情
- 事件与触发
- 物品 / 奖励发放

## 目录说明

- `官方/`
  存放官方公告、官方更新和官方能力边界摘要。
- `社区与开源线索/`
  存放教程、开源示例、可借鉴 mod 线索。

## 当前结论

- 官方已经明确支持：
  - 模组编辑器
  - 自定义奇遇 / 剧情方向
  - 配置表修改
  - 代码编写
  - 资源修改
  - NPC 互动剧情与道具相关条件 / 修复
- 当前最值得优先验证的鬼谷八荒实现路径不是“NPC 背包中转”，而是：
  - `剧情 / 事件触发 -> 产出物品 -> 玩家直接获得`
- 公开的标准 GitHub / Gitee 仓库线索不算多。
  当前最强线索来自：
  - 百度贴吧附件里的 `API 文档 + AI 奇遇开源代码`
  - 标注“开源版”的剧情 mod
  - 处理 NPC 发材料 / 背包格子的功能 mod
