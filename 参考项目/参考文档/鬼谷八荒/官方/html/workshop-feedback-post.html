【Workshop Feedback Post】
Dear friends, Steam Workshop is Available now!
We are in the process of optimising the mod function and workshops based on the suggestions you have made and the bugs/issues you have encountered in the game.
Workshop Entrance : Please click on Gmae - [Community Hub] - [Workshop] to enter, if it doesn't show up, please update the game and restart steam

If you encounter problems when using the mod editor or the workshop function, you can tell us in this thread, and we also kindly ask you to note the version number and the situation so that we can quickly locate the problem.

Here are some solutions to common problems
If you encounter an error:2, a common unknown error caused by network and other reasons, such as
Network error
Operation timeout, please try again!

Solutions:
(1) Please re-upload a few more times
(2) Family Sharing account cannot use the Workshop, you need to purchase the game to use it
(3) Restart your computer and try again

If you encounter a problem that we may not be able to recognize, we will contact you directly. You can also send your files to info@lightning.games, please note your Steam nickname and bug description.
Please note that we may not be able to fix bugs that are not created by the official mod editor, so we recommend contacting the mod creator first.
