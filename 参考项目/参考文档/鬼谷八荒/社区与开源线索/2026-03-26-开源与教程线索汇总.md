# 2026-03-26 开源与教程线索汇总

本文件汇总当前最值得继续深挖的鬼谷八荒开源 / 教程 / 社区实现线索。

## 结论先说

- 目前最强的不是标准 GitHub / Gitee 仓库，而是：
  - 百度贴吧附件里的 `API 文档 + AI奇遇开源代码`
  - 标注“开源版”的剧情 mod
  - 处理 NPC 发材料 / 背包格子的功能 mod
- 对“最简单发奖路径”的判断是：
  - `剧情 / 事件触发 -> 产出物品 -> 玩家直接获得`
  - 这条路比 `NPC 背包中转 -> 再赠送` 更短、更稳

## 优先级最高的线索

### 1. 百度贴吧：代码 Mod 零基础教程

- 链接：
  https://tieba.baidu.com/p/9739343547
- 已知信息：
  - 发帖人声称附件里包含：
    - 图文教程
    - API 接口文档
    - AI 奇遇 mod 开源代码
- 价值：
  - 最可能直接补齐我们现在最缺的“代码级实现细节”

### 2. NPC 自动突破（优化版）开源

- 链接：
  https://www.yufeiye.com/wp/5310
- 可确认事实：
  - 过月触发
  - 给亲族好友 NPC 发送突破材料
  - 赠送材料时增加 NPC 背包格子
  - 还会赋予对应戒指
- 能借鉴的点：
  - 定时触发
  - 关系过滤
  - 物品发放
  - 背包相关操作

### 3. 倒霉仙子总被强迫双修 开源版

- 链接：
  https://www.yufeiye.com/wp/5312
- 可确认事实：
  - 支持过月 / 进城 / 进酒馆 / 逃跑后触发
  - 有分支剧情和选择
  - 页面明确写“完全开源，可直接导入 mod 编辑器中修改”
- 能借鉴的点：
  - 多触发源映射
  - 分支对话
  - 编辑器可导入结构

### 4. Mod 制作拓展

- 链接：
  https://www.yufeiye.com/wp/5300
- 可确认事实：
  - 能增加交互按钮
  - 能合并剧情选项
  - 能播放奇遇剧情视频
  - 能替换剧情配图
- 能借鉴的点：
  - 对话入口扩展
  - 奇遇剧情演出
  - 更复杂的 UI / 演出能力

## 额外教程线索

### 产出物品命令

- 链接：
  https://www.bilibili.com/opus/761137890889564162
- 已知摘要：
  - 在命令中点击 `产出物品`
  - 每个命令只能给予玩家一种道具
  - 如设置多种物品，会按权重随机给其中一种
- 价值：
  - 这是当前最直接支持“最简单发奖路径”的线索
