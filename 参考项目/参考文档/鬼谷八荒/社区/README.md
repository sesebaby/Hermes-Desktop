# 鬼谷八荒社区模组线索摘要

更新时间：2026-03-26

目标：只保留和当前需求最相关的线索，重点看 `对话 / 组队 / NPC 行为 / 事件触发 / 物品发放`。

## 1. NPC-AI行动交互优化升级

- 原始页：`npc-ai-interaction-upgrade.html`
- 来源：https://steamcommunity.com/sharedfiles/filedetails/?id=2991543623&l=schinese

页面描述要点：

- 在好友 NPC 信息界面添加 `私人对话` 按钮
- NPC 每月更主动修炼和突破
- NPC 和玩家每月会尝试自动提升功法境界和品质

借鉴价值：

- 证明鬼谷八荒社区已经有人把“NPC 信息界面 -> 私人对话 -> NPC 行为变化”串起来了。
- 对我们最有用的是：
  - NPC 私人对话入口
  - 过月触发的 NPC 行为
  - 将 UI 按钮与 NPC 逻辑绑定

## 2. 灵活组队

- 原始页：`flexible-team-up.html`
- 来源：https://steamcommunity.com/sharedfiles/filedetails/?id=3031782868&l=schinese

页面描述要点：

- 需要前置 `Mod制作拓展`
- 大世界按 `T` 打开队伍成员列表
- NPC 面板左边新增 `组队` 按钮

借鉴价值：

- 证明可以改 NPC 面板交互按钮，并扩展多人关系和队伍 UI。
- 对我们最有用的是：
  - 多 NPC 交互入口
  - 群体交互管理方式
  - 前置框架 `Mod制作拓展` 的参考价值

## 3. 修真聊天群 / 诸天万界（联机mod）

- 原始页：`cultivation-chat-group.html`
- 来源：https://steamcommunity.com/sharedfiles/filedetails/?id=2958503186&l=schinese

页面描述要点：

- 回车开启 / 关闭 UI 或发送聊天信息
- 聊天窗口点击人名弹出交互菜单
- 提供透明模式、锁定 UI 等交互

借鉴价值：

- 证明鬼谷八荒里确实有人做出过聊天 UI。
- 虽然这是更重的联机聊天方向，但对我们最有用的是：
  - 对话 / 群聊 UI 的呈现思路
  - 点击头像 / 人名弹出交互菜单的方式

## 4. 更有趣的双修

- 原始页：`interesting-double-cultivation.html`
- 来源：https://steamcommunity.com/sharedfiles/filedetails/?id=3691229178&l=schinese

页面描述要点：

- 标签含 `气运 / 功能 / 道具 / 开源`
- 作者明确写了：`除代码以外，模组已开源`，修改模组或做附属模组需阅读 `ModAssets/README.MD`
- NPC 判定因素包含：性格、好感度、实力、魅力、亲族关系
- 支持怀孕、新生儿、好感相关行为和赠送行为

借鉴价值：

- 虽然不是完全源码开源，但它给出了一条很重要的线：
  - `ModAssets/README.MD` 可作为社区可读规则层
  - NPC 关系、判定、赠送行为已经被做出来过
- 对当前任务最有价值的是：
  - 关系 / 好感 / 事件驱动的 NPC 行为判定
  - “赠送”已经是社区里被用过的交互方向

## 5. 其他值得跟进的线索

- `官方灵田一键收获种植`
  - 在线索页中明确写了：`所用官方源码为 2022-06-21`，并且采用类重写方式，只重写 `FarmBattle` 类。
  - 这说明至少有社区作者直接基于官方示例源码二次改造。

- `击杀npc物品全掉落`
  - 说明物品掉落链路是有现成可改的。
  - 还提到可搭配另一个模组 `可赠送灵石给NPC`。

- `妻子道侣自动突破`
  - 描述里是按好感度排序，过月给 NPC 发突破材料与储物戒。
  - 这对“时间 / 好感度 / 过月 -> 送物品”非常贴近我们要的模型。
